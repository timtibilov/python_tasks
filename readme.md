# Гайды по заданиям

## Как установить Jupyter Notebook

1. Проверить на наличие `pip` на ноуте командой в терминале 
```bash
$ python3 -m pip --version
```
Должно вывести что-то похожее:
```bash
pip 22.3.1 from /usr/local/lib/python3.9/site-packages/pip (python 3.9)
```

2. Устанавливаем Jupyter Lab и Jupyter Notebook командой 
```bash
$ python3 -m pip install jupyterlab notebook
``` 

3. Заходим в папку с заданием в терминале и запускаем команду
```bash
$ python3 -m jupyter-lab
```

4. Автоматически должна открыться страница браузера, если это не произошло, переходим по [ссылке](http://localhost:8888/).
При этом в самом терминале запустится сервер. __Не закрывай окно терминала.__

5. В самом Jupyter Lab выбираешь нужный файл и работаешь с ним.

6. Чтобы выключить сервер, сохрани все свои изменения в файлах, чтобы не потерять, через `Cmd + S`, и нажми в терминале `Control + C`. Сервер предложит вырубится, напиши `y`, и сервер вырубится. Выглядит примерно так:
```bash
$ Shutdown this Jupyter server (y/[n])? y
```

7. Чтобы запустить сервер снова, переходи к пункту __3__.
